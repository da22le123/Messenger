#### Hi! This is a Messenger application implementation of the custom communication protocol using native sockets in Java.

### The provided functionality:

- Group chat with all connected users
- Direct messages
- Request all connected users
- Rock paper scissors game with a chosen user
- Tic tac toe game with a chosen user
- File trasnfer

### Implementation

Please, refer to the [protocol specification](/doc/protocol.md).

### Server usage

Feel free to use the server in order to build your own client application of any kind. 
Server is restricted to socket communication, other than that there are no limitation to what your client application uses.
